## v2.0.0 - 3D 可视化（计划：2026-04-01）

### 目标

集成 Three.js 3D 渲染引擎

### 功能清单

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 3D 场景渲染 | Three.js WebGL 渲染 | P0 |
| 场景控制 | 相机、灯光、材质控制 | P0 |
| 3D 文件支持 | OBJ/GLTF 模型加载 | P1 |
| 3D 导出 | 场景截图、模型导出 | P2 |

### 技术栈

```typescript
// 3D 模块

import { OrbitControls, Environment } from '@react-three/drei'

```
